# BCN3D_Moveo_ROS
ROS package para simulação de um modelo Moveo ds BCN3D.
1. Instalale o pacote:
-``` 
cd <your_workspace>/src
```
-```
git clone https://github.com/marcos3939/BCN3D_Moveo_ROS.git
```
-```
catkin_make
```
-```
cd ..
```
